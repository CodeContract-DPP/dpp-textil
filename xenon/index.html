Este demo esta alineado con ESPR (UE) 2024/1781, CIRPASS-2, Battery Regulation (UE) 2023/1542 como modelo analogo y Trace4Value como referencia textil. El acto delegado textil no esta adoptado a fecha de generacion de este demo (estimado mid-2028 segun JRC). Las secciones y campos son propuesta indicativa de Code Contract, no requisitos vinculantes. Los datos son ficticios para fines de demostracion.

Este HTML representa el Dataset DPP estructurado y firmado por Trackline (Code Contract), paso 3 del pipeline. Los pasos 1-2 (Fuentes documentales y motor Trackline) se simulan en este demo a partir de PDFs ficticios y JSONs gemelos. Los pasos 4-5 (Gobernanza por roles y exposicion via QR al consumidor) son fases siguientes del flujo Code Contract, fuera del alcance de esta vista.
